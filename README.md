# 🏠 Domestic Workers Management System

> **"Every great journey starts with a single line of code."** > This project is a special milestone for me—it's the very **first** application I ever wrote!

---

### ✨ About the Project
This is a simple Java-based application designed to help heads of families manage domestic workers' data. It was my first dive into the world of logic and programming, focused on:
* Tracking worker information.
* Managing monthly salaries and tips.
* Organizing basic schedules and profiles.

---

### 🌱 The "First Code" Story
I keep this repository here as a reminder of where it all began. While my skills have grown into **Flutter, React, and AI**, this project taught me the fundamentals of:
* Object-Oriented Programming (OOP) in Java.
* Logic building and problem-solving.
* Data structure basics.
